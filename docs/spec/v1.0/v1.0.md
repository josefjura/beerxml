# BeerXML 1.0 Specification

*This document is a reproduction of the original BeerXML 1.0 standard found at [beerxml.com](https://www.beerxml.com/beerxml.htm).*

## 1. Introduction

### Purpose
BeerXML is a free, fully defined XML data description for exchanging beer brewing recipes and other brewing data. It was designed to allow recipes to be exchanged between different brewing programs, and also to allow recipes to be posted on the web and read by brewing programs.

### General
*   **XML Standard:** All BeerXML files must be 100% compliant with the W3C XML standard.
*   **Encoding:** The default encoding is ISO-8859-1 (Latin-1). UTF-8 is also supported and recommended for international compatibility.
*   **Case Sensitivity:** XML tags are case sensitive. All BeerXML tags are strictly **UPPER CASE**.
*   **Versioning:** The current version is 1.0. All major records include a `<VERSION>` tag which must be set to `1` or `1.0`.

## 2. Formats and Units

Unless otherwise specified, all BeerXML 1.0 files use the following standard units. Programmers must convert to these units when reading/writing files.

### Units
| Measurement | Unit | Notes |
| :--- | :--- | :--- |
| **Weight** | Kilograms (kg) | Example: `1.5` (= 1.5 kg) |
| **Volume** | Liters (l) | Example: `20.0` (= 20 liters) |
| **Temperature** | Celsius (°C) | Example: `68.0` (= 68 °C) |
| **Time** | Minutes | Example: `60.0` (= 60 min) |
| **Pressure** | Kilopascals (kPa) | Used for carbonation. |
| **Acidity** | pH | |
| **Color** | SRM | Standard Reference Method |
| **Specific Gravity** | 1.000 format | Relative to water at 4°C |

### Data Formats
*   **Record Set:** A list of records (e.g., `<HOPS>...</HOPS>`).
*   **Record:** A single item (e.g., `<HOP>...</HOP>`).
*   **Text:** Standard character data.
*   **Integer:** Whole numbers (e.g., `5`).
*   **Boolean:** `TRUE` or `FALSE`.
*   **Percentage:** Expressed as a percentage, not a decimal (e.g., `5.5` for 5.5%, not 0.055).
*   **List:** A distinct set of text options (e.g., "Boil", "Mash").

## 3. Record Specifications

### Hop Record (`<HOP>`)

Used within a `<HOPS>` record set.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the hop. |
| `VERSION` | Yes | Integer | Version of the standard (Should be 1). |
| `ALPHA` | Yes | Percentage | Alpha acid percent (e.g., `5.5`). |
| `AMOUNT` | Yes | Weight (kg) | Amount of hops used in Kilograms. |
| `USE` | Yes | List | `Boil`, `Dry Hop`, `Mash`, `First Wort`, `Aroma`. Note: `Aroma` is usually for steep/whirlpool. |
| `TIME` | Yes | Time (min) | Time in minutes (boil time or dry hop time). |
| `NOTES` | No | Text | General notes. |
| `TYPE` | No | List | `Bittering`, `Aroma`, `Both`. |
| `FORM` | No | List | `Pellet`, `Plug`, `Leaf`. |
| `BETA` | No | Percentage | Beta acid percent. |
| `HSI` | No | Percentage | Hop Stability Index. |
| `ORIGIN` | No | Text | Place of origin (e.g., "UK", "US"). |
| `SUBSTITUTES` | No | Text | Suggested substitutes. |
| `HUMULENE` | No | Percentage | Humulene level. |
| `CARYOPHYLLENE` | No | Percentage | Caryophyllene level. |
| `COHUMULONE` | No | Percentage | Cohumulone level. |
| `MYRCENE` | No | Percentage | Myrcene level. |

### Fermentable Record (`<FERMENTABLE>`)

Used within a `<FERMENTABLES>` record set. Includes grains, extracts, and sugars.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the fermentable. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `TYPE` | Yes | List | `Grain`, `Sugar`, `Extract`, `Dry Extract`, `Adjunct`. |
| `AMOUNT` | Yes | Weight (kg) | Amount in Kilograms. |
| `YIELD` | Yes | Percentage | Percent yield (dry basis) or efficiency. e.g., `78.0`. |
| `COLOR` | Yes | Color (SRM) | Color in SRM. |
| `ADD_AFTER_BOIL`| No | Boolean | `TRUE` if added after the boil (late addition). Default `FALSE`. |
| `ORIGIN` | No | Text | Country or place of origin. |
| `SUPPLIER` | No | Text | Supplier or brand. |
| `NOTES` | No | Text | Notes. |
| `COARSE_FINE_DIFF`| No | Percentage | Difference between fine and coarse grind yield. |
| `MOISTURE` | No | Percentage | Moisture content. |
| `DIASTATIC_POWER`| No | Lintner | Diastatic power in Lintner degrees. |
| `PROTEIN` | No | Percentage | Protein content. |
| `MAX_IN_BATCH` | No | Percentage | Recommended max percentage in a grain bill. |
| `RECOMMEND_MASH`| No | Boolean | `TRUE` if mashing is recommended. |
| `IBU_GAL_PER_LB`| No | Float | For extracts: IBU contribution per gallon per pound. |

### Yeast Record (`<YEAST>`)

Used within a `<YEASTS>` record set.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the yeast. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `TYPE` | Yes | List | `Ale`, `Lager`, `Wheat`, `Wine`, `Champagne`. |
| `FORM` | Yes | List | `Liquid`, `Dry`, `Slant`, `Culture`. |
| `AMOUNT` | Yes | Weight/Vol | Amount of yeast (liters for liquid/starter, kg for dry). |
| `AMOUNT_IS_WEIGHT`| No | Boolean | `TRUE` if amount is weight (kg), `FALSE` if volume (liters). |
| `LABORATORY` | No | Text | Lab name (e.g., "Wyeast"). |
| `PRODUCT_ID` | No | Text | Product ID (e.g., "1056"). |
| `MIN_TEMPERATURE` | No | Temp (°C) | Min optimum temperature. |
| `MAX_TEMPERATURE` | No | Temp (°C) | Max optimum temperature. |
| `FLOCCULATION` | No | List | `Low`, `Medium`, `High`, `Very High`. |
| `ATTENUATION` | No | Percentage | Apparent attenuation percent. |
| `NOTES` | No | Text | Notes. |
| `BEST_FOR` | No | Text | Styles this yeast is best suited for. |
| `TIMES_CULTURED`| No | Integer | Number of times cultured/reused. |
| `MAX_REUSE` | No | Integer | Recommended max reuse count. |
| `ADD_TO_SECONDARY`| No | Boolean | `TRUE` if added for secondary fermentation. |

### Misc Record (`<MISC>`)

Used within a `<MISCS>` record set. For non-fermentables (Irish Moss, Spices, etc.).

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the item. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `TYPE` | Yes | List | `Spice`, `Fining`, `Water Agent`, `Herb`, `Flavor`, `Other`. |
| `USE` | Yes | List | `Boil`, `Mash`, `Primary`, `Secondary`, `Bottling`. |
| `TIME` | Yes | Time (min) | Time in minutes used. |
| `AMOUNT` | Yes | Weight/Vol | Amount. |
| `AMOUNT_IS_WEIGHT`| No | Boolean | `TRUE` if weight (kg), `FALSE` if volume (liters). |
| `USE_FOR` | No | Text | Description of use (e.g., "Clarity"). |
| `NOTES` | No | Text | Notes. |

### Water Record (`<WATER>`)

Used within a `<WATERS>` record set.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the profile. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `AMOUNT` | Yes | Volume (l) | Volume of water. |
| `CALCIUM` | Yes | ppm | Calcium (Ca) in parts per million. |
| `BICARBONATE` | Yes | ppm | Bicarbonate (HCO3) in ppm. |
| `SULFATE` | Yes | ppm | Sulfate (SO4) in ppm. |
| `CHLORIDE` | Yes | ppm | Chloride (Cl) in ppm. |
| `SODIUM` | Yes | ppm | Sodium (Na) in ppm. |
| `MAGNESIUM` | Yes | ppm | Magnesium (Mg) in ppm. |
| `PH` | No | pH | pH of the water. |
| `NOTES` | No | Text | Notes. |

### Style Record (`<STYLE>`)

Used within a `<STYLES>` record set.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the style. |
| `CATEGORY` | Yes | Text | Category name (e.g., "English Pale Ale"). |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `CATEGORY_NUMBER` | Yes | Text | Category number (e.g., "8"). |
| `STYLE_LETTER` | Yes | Text | Style letter (e.g., "A"). |
| `STYLE_GUIDE` | Yes | Text | Name of the guide (e.g., "BJCP 2004"). |
| `TYPE` | Yes | List | `Lager`, `Ale`, `Mead`, `Wheat`, `Mixed`, `Cider`. |
| `OG_MIN` | Yes | Gravity | Minimum Original Gravity. |
| `OG_MAX` | Yes | Gravity | Maximum Original Gravity. |
| `FG_MIN` | Yes | Gravity | Minimum Final Gravity. |
| `FG_MAX` | Yes | Gravity | Maximum Final Gravity. |
| `IBU_MIN` | Yes | IBU | Minimum IBU. |
| `IBU_MAX` | Yes | IBU | Maximum IBU. |
| `COLOR_MIN` | Yes | Color | Minimum Color (SRM). |
| `COLOR_MAX` | Yes | Color | Maximum Color (SRM). |
| `CARB_MIN` | No | Vol | Min Carbonation (Volumes of CO2). |
| `CARB_MAX` | No | Vol | Max Carbonation. |
| `ABV_MIN` | No | Percentage | Min Alcohol by Volume. |
| `ABV_MAX` | No | Percentage | Max Alcohol by Volume. |
| `NOTES` | No | Text | Description of the style. |
| `PROFILE` | No | Text | Flavor/Aroma profile. |
| `INGREDIENTS` | No | Text | Typical ingredients. |
| `EXAMPLES` | No | Text | Commercial examples. |

### Equipment Record (`<EQUIPMENT>`)

Used within an `<EQUIPMENTS>` record set, or nested in a recipe.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the equipment profile. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `BOIL_SIZE` | Yes | Volume (l) | Volume of the wort before the boil. |
| `BATCH_SIZE` | Yes | Volume (l) | Target volume after boiling/cooling. |
| `TUN_VOLUME` | No | Volume (l) | Volume of the mash tun. |
| `TUN_WEIGHT` | No | Weight (kg) | Weight of the mash tun (for heat calculation). |
| `TUN_SPECIFIC_HEAT`| No | Float | Specific heat of tun material (Cal/gram-deg C). |
| `TOP_UP_WATER` | No | Volume (l) | Water added to fermenter. |
| `TRUB_CHILLER_LOSS`| No | Volume (l) | Volume lost to trub/chiller. |
| `EVAP_RATE` | No | Percentage | Evaporation rate per hour of boil. |
| `BOIL_TIME` | No | Time (min) | Normal boil time. |
| `CALC_BOIL_VOLUME`| No | Boolean | `TRUE` if boil volume is calculated automatically. |
| `LAUTER_DEADSPACE`| No | Volume (l) | Volume lost in lauter tun deadspace. |
| `TOP_UP_KETTLE` | No | Volume (l) | Water added to kettle before boil. |
| `HOP_UTILIZATION` | No | Percentage | Efficiency adjustment for hop utilization (100% is normal). |
| `NOTES` | No | Text | Notes. |

### Mash Step Record (`<MASH_STEP>`)

Used within a `<MASH_STEPS>` record set.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the step. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `TYPE` | Yes | List | `Infusion`, `Temperature`, `Decoction`. |
| `INFUSE_AMOUNT` | No | Volume (l) | Volume of water to infuse (for Infusion steps). |
| `STEP_TEMP` | Yes | Temp (°C) | Target temperature for this step. |
| `STEP_TIME` | Yes | Time (min) | Duration of the step. |
| `RAMP_TIME` | No | Time (min) | Time to reach the step temperature (rise time). |
| `END_TEMP` | No | Temp (°C) | Temperature at the end of the step (for decoction/temp steps). |
| `DESCRIPTION` | No | Text | Text description of the step. |
| `WATER_GRAIN_RATIO`| No | Float | Ratio of water to grain (qt/lb or l/kg, usually informational). |
| `DECOCTION_AMT` | No | Volume (l) | Amount to decoct (for Decoction steps). |
| `INFUSE_TEMP` | No | Temp (°C) | Temperature of the infusion water. |

### Mash Profile Record (`<MASH>`)

Used within a `<MASHS>` record set, or nested in a recipe. Contains `<MASH_STEPS>`.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the profile. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `GRAIN_TEMP` | Yes | Temp (°C) | Initial grain temperature. |
| `MASH_STEPS` | Yes | Record Set | List of `<MASH_STEP>` records. |
| `NOTES` | No | Text | Notes. |
| `TUN_TEMP` | No | Temp (°C) | Initial tun temperature. |
| `SPARGE_TEMP` | No | Temp (°C) | Target sparge temperature. |
| `PH` | No | pH | Target mash pH. |
| `TUN_WEIGHT` | No | Weight (kg) | Weight of tun (overrides Equipment). |
| `TUN_SPECIFIC_HEAT`| No | Float | Specific heat (overrides Equipment). |
| `EQUIP_ADJUST` | No | Boolean | `TRUE` if software should adjust temps for equipment thermal mass. |

### Recipe Record (`<RECIPE>`)

Used within a `<RECIPES>` record set. This is the top-level record for most files.

| Tag | Req? | Format | Description |
| :--- | :--- | :--- | :--- |
| `NAME` | Yes | Text | Name of the recipe. |
| `VERSION` | Yes | Integer | Version (Should be 1). |
| `TYPE` | Yes | List | `Extract`, `Partial Mash`, `All Grain`. |
| `STYLE` | No | Record | The `<STYLE>` record for this recipe. |
| `EQUIPMENT` | No | Record | The `<EQUIPMENT>` record used. |
| `BREWER` | Yes | Text | Brewer's name. |
| `ASST_BREWER` | No | Text | Assistant brewer's name. |
| `BATCH_SIZE` | Yes | Volume (l) | Target size of finished batch. |
| `BOIL_SIZE` | Yes | Volume (l) | Starting size of boil. |
| `BOIL_TIME` | Yes | Time (min) | Boil duration. |
| `EFFICIENCY` | No | Percentage | Brewhouse efficiency (e.g., `72.0`). |
| `HOPS` | No | Record Set | List of `<HOP>` records. |
| `FERMENTABLES` | No | Record Set | List of `<FERMENTABLE>` records. |
| `MISCS` | No | Record Set | List of `<MISC>` records. |
| `YEASTS` | No | Record Set | List of `<YEAST>` records. |
| `WATERS` | No | Record Set | List of `<WATER>` records. |
| `MASH` | No | Record | The `<MASH>` profile used. |
| `NOTES` | No | Text | Recipe notes. |
| `TASTE_NOTES` | No | Text | Tasting notes. |
| `TASTE_RATING` | No | Float | Rating (e.g., 0-50). |
| `OG` | No | Gravity | Measured Original Gravity. |
| `FG` | No | Gravity | Measured Final Gravity. |
| `FERMENTATION_STAGES`| No | Integer | Number of fermentation stages. |
| `PRIMARY_AGE` | No | Time (days)| Time in primary. |
| `PRIMARY_TEMP` | No | Temp (°C) | Temperature in primary. |
| `SECONDARY_AGE` | No | Time (days)| Time in secondary. |
| `SECONDARY_TEMP` | No | Temp (°C) | Temperature in secondary. |
| `TERTIARY_AGE` | No | Time (days)| Time in tertiary. |
| `TERTIARY_TEMP` | No | Temp (°C) | Temperature in tertiary. |
| `AGE` | No | Time (days)| Time in aging/bottle. |
| `AGE_TEMP` | No | Temp (°C) | Temperature for aging. |
| `DATE` | No | Text | Brewing date. |
| `CARBONATION` | No | Vol | Carbonation (volumes of CO2). |
| `FORCED_CARBONATION`| No | Boolean | `TRUE` if forced carbonated. |
| `PRIMING_SUGAR_NAME`| No | Text | Name of priming sugar. |
| `CARBONATION_TEMP`| No | Temp (°C) | Temperature for carbonation. |
| `PRIMING_SUGAR_EQUIV`| No | Float | Conversion factor to corn sugar (1.0 = corn sugar). |
| `KEG_PRIMING_FACTOR`| No | Float | Factor for kegging (usually 0.5 of bottling amount). |

## Appendix A: Optional Extensions for Display

These tags are optional and used primarily for displaying converted units or calculated values.

| Tag | Description |
| :--- | :--- |
| `EST_OG` | Estimated Original Gravity |
| `EST_FG` | Estimated Final Gravity |
| `EST_COLOR` | Estimated Color |
| `IBU` | Estimated IBU (Bitterness) |
| `IBU_METHOD` | IBU Formula (`Rager`, `Tinseth`, `Garetz`) |
| `EST_ABV` | Estimated Alcohol by Volume |
| `ABV` | Measured Alcohol by Volume |
| `ACTUAL_EFFICIENCY` | Calculated efficiency based on OG/FG |
| `CALORIES` | Calories per serving |
| `DISPLAY_BATCH_SIZE` | Batch size with units (e.g. "5.0 gal") |
| `DISPLAY_BOIL_SIZE` | Boil size with units |
| `DISPLAY_OG` | OG with units |
| `DISPLAY_FG` | FG with units |
| `DISPLAY_PRIMARY_TEMP`| Primary Temp with units |
| `DISPLAY_SECONDARY_TEMP`| Secondary Temp with units |
| `DISPLAY_TERTIARY_TEMP`| Tertiary Temp with units |
| `DISPLAY_AGE_TEMP` | Age Temp with units |
| `CARBONATION_USED` | Description of carbonation used |
| `DISPLAY_CARB_TEMP` | Carbonation temp with units |
